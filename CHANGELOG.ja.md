# 変更履歴

MDViewer の主な変更はこのファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) に準拠しています。

---

## [1.1.1] — 2026-06-12

### 修正
- Markdown ファイルと同じディレクトリにあるローカル画像が正しく表示されるようになりました。相対パスの画像を `file://` ではなく `mdviewer-local://` スキームハンドラ経由で配信するようにし、WebView のサンドボックスによるブロック（画像がリンク切れに見える問題）を解消しました。

---

## [1.1.0] — 2026-05-22

### 追加
- 分割ビューエディタモード: ⌘E またはツールバーの鉛筆ボタンで切り替え
- 左ペインに等幅フォントのエディタ、右ペインにリアルタイムプレビューを表示
- 保存機能: ⌘S で現在のファイルを上書き保存（read-write Sandbox 権限を追加）
- 未保存変更ガード: 未保存の変更がある状態でウィンドウを閉じると「保存・破棄・キャンセル」の確認ダイアログを表示

---

## [1.0.3] — 2026-05-06

### 追加
- エクスポート: デフォルトファイル名が元の Markdown ファイル名を継承するように変更（例: `document.pdf` → `README.pdf`）。日本語・スペースを含むファイル名のパーセントエンコードも正しくデコード
- タイトルバーに「MDViewer」ではなく開いているファイル名を表示するように変更

### 削除
- ページサムネイルサイドバーを削除。サイドバーは目次（TOC）のみに

---

## [1.0.2] — 2026-05-05

### 変更
- PDF エクスポート: `WKWebView.createPDF()`（単一ページ）を `NSPrintOperation.runModal` に変更し、印刷用 CSS を正しく適用して複数ページの PDF を生成するように修正

### 修正
- `NSPrintOperation.runModal` への切り替えにより PDF エクスポートのハングアップを解消
- サイドバー: サムネイルタブを非表示（目次のみ表示）

---

## [1.0.1] — 2026-05-05

### 追加
- PDF・印刷レイアウト: A4 サイズ・マージン・改ページ制御の `@media print` スタイルを追加
- ビルド・公証スクリプト（`build-notarize.sh`）を追加

### 修正
- Shiki シンタックスハイライタ: フォールバック処理と 8 秒の初期化タイムアウトを追加
- Shiki 向けの冗長なライトテーマ CSS オーバーライドを削除（インラインスタイルが優先されるため）

---

## [1.0.0] — 2026-05-04

### 追加
- 初回リリース
- WKWebView + marked.js v12 による Markdown レンダリング
- Shiki v1 による 27 言語のシンタックスハイライト（github-light / github-dark デュアルテーマ）
- KaTeX v0.16 による LaTeX 数式表示 — インライン `$…$` およびブロック `$$…$$`
- Mermaid 図のサポート — フローチャート、シーケンス図、ガントチャート
- 見出しから自動生成される目次サイドバー
- `DispatchSource`（kqueue、0.5 秒デバウンス）によるファイル自動リロード
- カスタム `mdviewer-local://` URL スキームハンドラによるローカル画像の読み込み
- テーマ切り替え — GitHub Light / GitHub Dark、macOS の外観設定に追従
- フォントサイズ調整 — 拡大・縮小・リセット（⌘+、⌘−、⌘0）
- PDF エクスポート
- HTML エクスポート
- スマートリンク処理 — ローカル `.md` リンクはアプリ内で開き、外部リンクはブラウザへ
- ページ内テキスト検索（⌘F）
- 日本語・英語ローカライズ
- Developer ID 署名と Apple 公証
- ランディングページ（英語・日本語）: https://masakai.github.io/mdviewer/
