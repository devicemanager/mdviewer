MDViewer — Privacy Policy

Last updated: 2026-07-17

MDViewer does not collect, store, transmit, or sell any personal data. There are no accounts, no analytics, no tracking, no advertising, and no telemetry of any kind.

What MDViewer does

MDViewer renders Markdown files entirely on your Mac. Parsing, syntax highlighting, math, and diagrams all run locally inside the app. The app has no servers and never sends your documents, filenames, or usage anywhere.

Files

MDViewer runs in the macOS App Sandbox and only accesses files you explicitly open, plus — with your permission — the folder of a document that references local images. It does not scan or read anything else on your disk.

Network access

By default MDViewer makes no network connections. It contacts the network only when a document you open references a remote (http/https) image and you allow it via Preferences → Privacy (the default is to ask). When you allow it, your Mac fetches those images directly from the servers that host them — the same as any web page loading an image — which discloses your IP address to those third-party servers. MDViewer itself receives nothing and stores nothing about this.

Third-party services

None. MDViewer bundles open-source rendering libraries (marked, Shiki, KaTeX, Mermaid, DOMPurify) that run locally; none of them phone home.

Children

MDViewer is rated 4+ and collects no data from anyone, including children.
